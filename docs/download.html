SEC PNG
iPhone/iPad: tap Open / Save PNG. If it opens the image preview, tap the green ✓ (top-right), then choose Save Image / Save to Photos.
iPhone / iPad — Save Steps

When the image preview opens:

  1. Tap the green check ✓ in the top-right
  2. Select Save Image (or Save to Photos)
✓ That ✓ is the iOS preview “Done” button.
Backup: if you only see the image, press-and-hold it and choose Save to Photos.
